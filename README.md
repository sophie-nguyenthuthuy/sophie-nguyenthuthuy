## What I build

I implement the systems behind modern data platforms **from scratch** — to understand them, and to teach them.

🏗️ **[data-engineering](https://github.com/sophie-nguyenthuthuy/data-engineering)** — my flagship: a monorepo of **93+ self-contained projects** (2,500+ tests, `mypy --strict`) covering streaming, CDC, table formats (Iceberg/Delta/Hudi internals), query engines, consensus (Raft, Flexible Paxos), CRDTs, and differential privacy. Start with the [featured five](https://github.com/sophie-nguyenthuthuy/data-engineering#-featured-projects).

🧠 **ML × databases** — [cardinality-learned-optimizer](https://github.com/sophie-nguyenthuthuy/cardinality-learned-optimizer): the Neo/Bao learned query-optimizer loop on PostgreSQL — TreeLSTM cardinality estimation, Thompson-sampling plan selection, JOB benchmark reproduction.

🏠 **Modern lakehouse, laptop-scale** — [practical-data-engineering-2026](https://github.com/sophie-nguyenthuthuy/practical-data-engineering-2026): dlt → Iceberg on MinIO → DuckDB → Dagster → Streamlit. The 2021 "20-minute data project," rebuilt with 2026 tooling — no Spark, no Kubernetes.

🇻🇳 **Vietnam-first products & teaching**
- [aec-platform](https://github.com/sophie-nguyenthuthuy/aec-platform) — AI SaaS for Architecture/Engineering/Construction: 16 modules from bid discovery to fire-safety compliance (Python + TypeScript, multi-tenant RLS, LangGraph).
